# 芋道云模块详细介绍

## 项目概述

芋道云（yudao-cloud）是一个基于 Spring Cloud Alibaba 微服务架构的快速开发平台，采用前后端分离的设计模式，集成了系统功能、基础设施、工作流程、支付系统、数据报表、会员中心、商城系统、微信公众号、CRM、ERP、AI大模型、物联网等多种业务功能。

## 核心架构模块

### yudao-dependencies
- **功能**：Maven 依赖版本管理模块
- **作用**：统一管理整个项目的所有依赖版本，确保各个模块使用统一的依赖版本，避免版本冲突问题。

### yudao-framework
- **功能**：Java 框架扩展模块
- **作用**：提供基础框架功能，包括：
  - 通用工具类
  - 安全框架扩展
  - 监控组件
  - 消息队列支持
  - 数据库连接池
  - 参数校验
  - 缓存支持
  - RPC调用等

### yudao-gateway
- **功能**：微服务网关模块
- **作用**：作为系统的统一入口，提供：
  - 请求路由
  - 负载均衡
  - 限流熔断
  - 认证鉴权
  - 跨域处理
  - 日志记录等

### yudao-server
- **功能**：管理后台 + 用户 APP 的服务端模块
- **作用**：整合各个业务模块，提供统一的服务入口，包含：
  - 管理后台API
  - 用户APP API
  - 认证授权服务
  - 系统配置等

## 业务功能模块

### yudao-module-system（系统功能模块）
- **功能**：系统核心功能管理
- **作用**：提供系统基础功能，包括：
  - 用户管理：管理系统的操作用户
  - 角色管理：分配菜单权限和操作权限
  - 菜单管理：配置系统菜单、操作权限、按钮权限
  - 部门管理：配置组织机构，支持树结构
  - 岗位管理：管理用户担任的职务
  - 租户管理：支持SaaS多租户功能
  - 字典管理：维护系统固定数据
  - 短信管理：对接短信平台
  - 操作日志：记录系统操作日志
  - 登录日志：记录用户登录日志
  - 错误码管理：管理系统的错误码
  - 通知公告：发布系统通知

### yudao-module-infra（基础设施模块）
- **功能**：基础设施管理
- **作用**：提供系统基础支撑功能，包括：
  - 代码生成：自动生成前后端代码
  - 系统接口：基于Swagger生成API文档
  - 数据库文档：生成数据库文档
  - 表单构建：拖拽生成表单代码
  - 配置管理：动态配置系统参数
  - 定时任务：基于XXL-Job的任务调度
  - 文件服务：支持多种文件存储方式
  - API日志：记录API访问和异常日志
  - 消息队列：基于Redis的消息队列
  - 监控功能：Java监控、链路追踪、日志中心
  - 服务保障：分布式锁、幂等、限流等

### yudao-module-member（会员中心模块）
- **功能**：会员管理
- **作用**：管理C端用户，包括：
  - 会员管理：管理会员信息
  - 会员标签：对会员进行标签分类
  - 会员等级：管理会员等级和成长值
  - 会员分组：对会员进行分组管理
  - 积分签到：积分系统管理

### yudao-module-bpm（工作流程模块）
- **功能**：工作流引擎
- **作用**：基于Flowable构建的工作流系统，包括：
  - BPMN设计器：标准工作流设计
  - 简易设计器：仿钉钉/飞书设计器
  - 流程模型管理：流程定义和部署
  - 任务管理：待办、已办任务处理
  - 流程表单：动态表单配置
  - 审批功能：支持会签、或签、依次审批等
  - 流程监控：流程实例监控

### yudao-module-pay（支付系统模块）
- **功能**：支付系统
- **作用**：提供多种支付渠道接入，包括：
  - 应用信息管理：配置支付渠道信息
  - 支付订单管理：处理支付订单
  - 退款订单管理：处理退款订单
  - 回调通知：处理支付结果通知
  - 支付接入示例：提供支付接入指导

### yudao-module-report（数据报表模块）
- **功能**：报表系统
- **作用**：提供报表和大屏功能，包括：
  - 报表设计器：支持数据报表、图形报表、打印设计
  - 大屏设计器：拖拽生成数据大屏
  - 图表组件：内置多种图表组件

### yudao-module-mp（微信公众号模块）
- **功能**：微信公众号管理
- **作用**：管理微信公众号功能，包括：
  - 账号管理：配置多个微信公众号
  - 数据统计：统计用户和消息数据
  - 粉丝管理：管理公众号粉丝
  - 消息管理：处理粉丝消息
  - 模版消息：发送模版消息
  - 自动回复：配置自动回复规则
  - 标签管理：管理公众号标签
  - 菜单管理：配置公众号菜单
  - 素材管理：管理公众号素材

### yudao-module-mall（商城系统模块）
- **功能**：电商系统
- **组成**：
  - yudao-module-product：商品管理模块
    - 商品分类
    - 商品信息管理
    - 商品属性管理
    - 商品规格管理
  - yudao-module-promotion：营销推广模块
    - 优惠券管理
    - 活动管理
    - 促销管理
  - yudao-module-trade：交易管理模块
    - 订单管理
    - 购物车
    - 支付管理
    - 物流管理
  - yudao-module-statistics：数据统计模块
    - 销售统计
    - 商品统计
    - 用户统计

### yudao-module-erp（ERP系统模块）
- **功能**：企业资源规划系统
- **作用**：企业管理相关功能，包括：
  - 采购管理
  - 销售管理
  - 库存管理
  - 财务管理
  - 供应链管理
  - 生产管理等

### yudao-module-crm（CRM系统模块）
- **功能**：客户关系管理系统
- **作用**：客户管理相关功能，包括：
  - 客户信息管理
  - 销售机会管理
  - 联系人管理
  - 商机跟进
  - 合同管理
  - 客户分析等

### yudao-module-ai（AI大模型模块）
- **功能**：人工智能模块
- **作用**：集成AI大模型功能，包括：
  - 大模型接入
  - AI对话功能
  - 智能助手
  - 文本生成
  - 智能分析等

### yudao-module-iot（物联网模块）
- **功能**：物联网系统
- **组成**：
  - yudao-module-iot-api：物联网API接口
  - yudao-module-iot-core：物联网核心功能
  - yudao-module-iot-gateway：物联网网关
  - yudao-module-iot-server：物联网服务端
- **作用**：提供物联网相关功能，包括：
  - 设备管理
  - 数据采集
  - 设备监控
  - 物联网协议支持
  - 数据分析等

## 模块关系

各个模块之间通过API模块进行通信，实现了良好的解耦。每个业务模块都包含API模块和Server模块：
- API模块：定义接口契约，供其他模块调用
- Server模块：实现具体的业务逻辑

这种设计使得系统具有良好的可扩展性和可维护性，可以根据业务需要灵活添加或移除模块。
